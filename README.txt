chamber: contains the alibre design files for the main turbidostat chamber
main_board: contains the eagle brd and sch files for the main controller board
main_board/code: contains the avr C code for the mainboard
pinchvalve: contains the alibre design files for the multiplex pinch valves.  (coming soon: drivers for solinoid pinch valves)
syring_pump: contains the alibre design files for the 3d printed syringe pump.  Comercial pumps may be used instead at added expense.
